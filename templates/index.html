Getting Started

Make some changes, then click "Update" to see the results. If you don't like it, click "Revert" to delete all changes.


All actions are additive i.e. increasing brightness by 49% then 51% is the same as doig 100%.


If you don't change a value between pressing "Update", then changes aren't calculated.


Some changes may take a few seconds to process.
